Vamos aprender HTML e CSS ?

Mas afinal o que é HTML ?

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. Então "ela" não é uma linguagem de programação.

E o que é CSS ?

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos. Normalmente o CSS é usado com o HTML, como nesse projeto !

E por onde começar ?

Para alguem que deseja começar a programar web um dos caminhos mais comuns a se seguir e ir para o HTML5 e o CSS3, que atualmente é usado por diversas empresas e sites ao redor do mundo, alem de seu nivel de complexidade nao ser muito auto.
